## 常见追问 / Follow-up Questions

### Q1: Tell me more about your staking project.

> The RNT staking project has three main contracts. First, RNT is an ERC20 token with ERC2612 permit standard. Second, esRNT is a vesting token that locks tokens and releases linearly over time. Third, StakePool manages staking, unstaking, and reward distribution. Users can stake RNT, earn rewards, and claim their earnings at any time.

### Q2: How do you interact with smart contracts from backend?

> I use ethers.js on the backend to interact with smart contracts. First, I create a Contract instance with the contract address and ABI. Then I call read functions directly. For write operations, I sign the transaction with a private key and send it to the blockchain. I also listen to contract events to keep the off-chain database in sync.
